## Access via @vercel/integration-utils

If you use the `@vercel/integration-utils` helper to create the UIHook, you can access the Metadata Store via the vercelClient instance, for example:

```js
const { withUiHook } = require('@vercel/integration-utils')

module.exports = withUiHook(async ({ payload, vercelClient }) => {
  // Get metadata
  const metadata = await vercelClient.getMetadata()
  metadata.count = metadata.count || 0
  metadata.count += 1

  // Set metadata
  await vercelClient.setMetadata(metadata)

  return `
    <Page>
    <P>Counter: ${metadata.count}</P>
      <Button>Count Me</Button>
    </Page>
  `
})
```
